03.locate_print

  他で作成したone_line_editerを使用して簡単なコマンド実行に対応させたものです。
  コマンドには
    cls
    exit
    exec
    help
  があります。execは入力した文字を表示するだけです。


  This uses the previously created one_line_editter to support simple command execution.
  The available commands are cls, exit, exec, and help.
  exec only displays the input string.
