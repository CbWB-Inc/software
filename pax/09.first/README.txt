09.first

  ここまでの集大成です。
  キーボードからコマンドを受け取り
  エンターで実行します。
  helpで実行できるコマンドが示されます。
  コマンドによりディスクの特定セクタをメモリに読み込み実行します。
  裏では2つのタスクが同時に実行されています。
  TSSではありませんがマルチタスクになっています。
  デモンストレーション用の成果物になるかと思います。


  This is the culmination of everything so far.
  Receives commands from the keyboard and executes them upon pressing Enter.
  The help command shows the executable commands.
  Loads a specific sector from disk into memory and executes it based on the command.
  Two tasks are running simultaneously in the background.
  It is multitasking, though not via TSS.
  This is meant as a demonstration artifact.
