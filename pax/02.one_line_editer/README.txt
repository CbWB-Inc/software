02.one_line_editter

  1行だけのエディタ機能を持ったechoプログラム
  左右方向キーでカーソル移動、通常キーで上書き
  リターン押下時に行頭からリターンまでを返す仕様です。
  BSキーとdelキーに対応できていないのが残念ですが
  状況により使い道はあると思います。


  An echo program with single-line editing functionality  
  Cursor moves with left/right keys, and normal keys overwrite characters  
  When Return is pressed, it returns content from line start to that point  
  It's unfortunate that the BS and Del keys are not supported  
  Still, it may have use depending on the situation
