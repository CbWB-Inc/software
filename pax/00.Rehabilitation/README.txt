# 00.Rehabilitation

This is a rehabilitation source for refreshing my memory and verifying the environment after about a year-long break.

## Features

- Moves the cursor using arrow keys only.
- Experiments with cursor positioning via BIOS interrupts.
- Cursor responds to:
  - Up
  - Down
  - Left
  - Right
