06.interuption

  タイマ割り込み処理の試作です。
  割り込み自体は機能していると思うのですが
  そもそもの意図通りには動いていまいと思われます。
  これをたたき台にして割り込み処理を中心にしたプログラムを
  組むようになります。
  おそらく失敗しているとはいえ記念碑的な内容です。


  A trial implementation of timer interrupt handling.
  The interrupt itself seems to be working.
  However, it probably doesn't behave as originally intended.
  This serves as a basis for later programs focused on interrupt handling.
  Though likely unsuccessful, it’s still a landmark piece.
