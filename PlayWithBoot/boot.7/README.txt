  This is a compilation of everything I’ve learned so far.

  Think of it as a showcase of what’s now possible  
  thanks to all the small steps taken along the way.

  I’ve bundled together the core routines I’ll likely need in the future,  
  along with simple test code for each.

  The tests are still pretty basic — there may be bugs hiding!  
  But I think it shows my general development style.

  I’m a “test-first” kind of person.  
  Writing tests first defines behavior precisely,  
  and once that’s done, implementation can be done at your own pace.

  (Though I do occasionally go full muscle-brained and code by pure instinct... 😅)

  Hopefully some of this proves helpful to others.
