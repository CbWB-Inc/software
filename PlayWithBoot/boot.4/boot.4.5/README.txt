boot.4.5

  I finally managed to conquer 5050! 🎉

  But as I was writing the program, I couldn’t help but feel a tickle —  
  in my brain… or maybe in my stomach.

  The same kind of processing… repeated again and again…  
  Slightly different each time, but so repetitive.  
  It felt… “tedious.”

  So I decided to improve it.

  Since it’s a 4-digit number, I could’ve just repeated the code 4 times.  
  But I wanted something better — something that works for 2 digits, 5 digits… anything.

  Here’s the idea:
  - Divide the number by 10
  - Keep dividing and printing the remainder
  - Once the quotient becomes 0, we’re done

  This way, it works for any number of digits.

  Now I can run experiments and check the results with confidence!

  Mission accomplished 😄
